A Smart Campus Resource Management System designed to simplify and automate campus operations using modern web technologies.

## 📌 Overview

CampusNexus AI is a full-stack web application that helps educational institutions efficiently manage campus resources such as:

- 📦 Inventory Management
- 🚚 Logistics Management
- 🛠 Maintenance Requests
- 📚 Library Management
- 👥 User Management
- 🤖 AI-based Demand Prediction
- 📝 Complaint Management

The application provides an intuitive dashboard for administrators to monitor and manage campus resources in real time.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/your-repository-link.git
```

---

## Backend Setup

```bash
cd CampusNexus-AI/backend
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run backend

```bash
python -m uvicorn app:app --reload
```

Backend URL

```
http://127.0.0.1:8000
```

API Documentation

```
http://127.0.0.1:8000/docs
```

---

## Frontend Setup

Open another terminal

```bash
cd CampusNexus-AI/frontend
```

Install packages

```bash
npm install
```

Run frontend

```bash
npm run dev
```

Frontend URL

```
http://localhost:5173
```
